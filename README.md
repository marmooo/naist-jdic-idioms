# naist-jdic-idioms
NAIST-jdic からニ文字熟語や三文字熟語を機械的に生成します。

## Installation
1. install [NAIST-jdic](https://ja.osdn.net/projects/naist-jdic/downloads/53500/mecab-naist-jdic-0.6.3b-20111013.tar.gz/)
2. ```npm install```

## Usage
```
node build.js
```
